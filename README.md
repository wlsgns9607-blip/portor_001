# 🛁 Bubble Mate Portfolio

**B2C 세차용품 전문 커머스 플랫폼, Bubble Mate의 포트폴리오 웹페이지입니다.**

세차 매니아들을 위한 직관적이고 빠른 쇼핑 경험을 제공하며, 프로페셔널한 디테일링 제품을 한 곳에서 만나볼 수 있는 'Bubble Mate'의 핵심 기능과 디자인 시스템을 소개하는 소개 페이지입니다.

---

## 🎯 프로젝트 개요
- **프로젝트명:** Bubble Mate 포트폴리오 웹페이지
- **기획 배경:** 2년간의 디테일링 세차 실무 경험과 자동차에 대한 깊은 이해를 바탕으로, 직접 잘 아는 도메인(세차 용품)을 타겟으로 기획했습니다. 단순한 화면 구현을 넘어 토스 페이먼츠 결제 API를 연동하여 실무에 가까운 커머스 플로우를 구축하는 데 중점을 두었습니다.
- **역할 및 협업:**
  - **UX/UI 디자인:** Figma를 활용한 와이어프레임 및 고피델리티 디자인 기획
  - **AI 협업 (Claude 등):** AI 비서(Claude)와 짝프로그래밍(Pair Programming)하며 HTML, CSS, JavaScript를 기반으로 프론트엔드 UI를 구현 및 고도화

## ✨ 핵심 기능 (Features Showcase)
Bubble Mate 커머스 플랫폼에 구현된 주요 기능들을 시각적으로 깔끔하게 정리했습니다.
1. **상품 상세 페이지:** 세차용품의 특성을 한눈에 파악할 수 있는 직관적인 UI/UX
2. **토스 페이먼츠 결제 연동:** 장바구니부터 실제 테스트 결제까지 이어지는 완벽한 이커머스 플로우
3. **관리자(Admin) 대시보드:** 쇼핑몰 운영자를 위한 직관적인 매출 통계 및 상품/회원 관리 기능
4. **멀티 소셜 로그인:** 다양한 소셜 연동을 통해 사용자의 회원가입 장벽을 낮추고 UX 향상
5. **직관적인 장바구니 (사이드바 UI):** 이탈률을 낮추기 위해 우측에서 부드럽게 열리는 결제 연동형 사이드바 장바구니 구현

## 🎨 디자인 시스템 & 인터랙션
사용자 경험(UX)을 극대화하고 프리미엄 세차 용품 플랫폼의 아이덴티티를 살리기 위해 **청록/블루 계열의 파스텔 톤 그라데이션**을 메인 디자인 테마로 채택했습니다.
- 통일감 있고 트렌디한 컬러 팔레트 (`#0ED2F7`, `#74EBD5`, `#B2FEFA` 등) 적용
- 요소별 그림자(Box-shadow) 및 떠오르는 호버(Hover) 애니메이션 인터랙션을 통한 동적 디자인
- 전체적인 레이아웃의 여백과 둥근 테두리(border-radius)를 활용한 부드러운 UI 구성

## 📱 반응형 웹 디자인
모든 기기(PC, Tablet, Mobile) 환경에서 완벽하게 작동하도록 반응형으로 설계되었습니다. 모바일 해상도(Galaxy Fold 등)에서도 깨짐 없이 레이아웃이 자연스럽게 재배치됩니다.
